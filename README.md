# Idea Pad

Ever had a sudden idea and wished you could jot it down instantly? **Idea Pad** does exactly that but smarter. Built in **Java**, it lets you **add, view, search, and delete ideas**, all safely saved in a file so nothing is ever lost.

Think of it as your **digital notebook**, ready whenever inspiration strikes! Simple, fast, and reliable.

---

## Features

- **Add Ideas:** Capture your thoughts in seconds  
- **View Ideas:** See all your stored notes at a glance  
- **Search Ideas:** Find ideas with keywords, no scrolling needed  
- **Delete Ideas:** Remove ideas when they’re no longer relevant  
- **Persistence:** Ideas stay saved in `ideas.txt` forever  

---

## Why This Project

**Idea Pad** helped me explore **Java basics**, **file handling**, and **dynamic data storage**. It’s a small CLI project that’s **practical, reusable, and perfect for learning programming logic**.
